# 🏟️ 스포츠 이탈 위험 조기탐지 대시보드 (Early Churn Detection)

본 프로젝트는 스포츠 스트리밍 서비스 유저의 시청 패턴을 분석하여 **이탈 위험군을 조기에 발견하고 유저 세그먼트별 맞춤형 액션 가이드**를 제공하는 대시보드입니다.

## 🚀 주요 기능
- **Overview**: 주차별 이탈 위험군 추이 및 핵심 KPI(재방문율, 시청 시간) 모니터링
- **세그먼트 분석**: 유저 유형별(열성팬, 하이라이트 위주 등) 위험도 분석
- **퍼널 분석**: 유지군과 위험군 간의 유입부터 재방문까지의 여정 비교
- **타이밍 분석**: 마지막 시청 후 경과 시간별 재방문 확률 및 골든타임 도출
- **액션 가이드**: A/B 테스트 결과 기반 세그먼트별 추천 액션 및 KPI 제시

## 🛠️ 실행 방법
1. 저장소 클론: `git clone [저장소 주소]`
2. 라이브러리 설치: `pip install -r requirements.txt`
3. 대시보드 실행: `streamlit run app.py`

## 📊 사용 데이터
- 가상으로 생성된 유저 시청 로그 및 이탈 위험 라벨링 데이터 (Sample Data)
